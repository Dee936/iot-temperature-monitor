Overview

What is ThermoControl IoT?

This project combines embedded hardware and mobile software into a smart environment monitoring system. A physical temperature sensor continuously reads the surroundings, sends data to a mobile app in real time, and triggers automatic fan control when temperature crosses a threshold — while giving full manual control via the app switch.

Problem
Fans and cooling systems need to be turned on manually even when automation is straightforward — wasting time and energy.
Solution
An IoT system with a sensor, microcontroller, and mobile app that automates fan control and shows live temperature readings.
Hardware
Temperature sensor + microcontroller (Arduino/ESP8266) + fan with relay switch module
Software
Mobile app with real-time temperature display and manual ON/OFF switch controls
System Workflow

How It Works

The system runs a continuous sense → transmit → display → respond loop entirely in real time.

1

Temperature Sensing

A DHT11/DHT22 temperature sensor attached to the hardware model continuously reads the surrounding ambient temperature at regular intervals.

DHT11 / DHT22 Sensor
2

Microcontroller Processing

The Arduino/ESP8266 microcontroller reads the sensor values, processes them, and transmits the temperature data wirelessly to the connected mobile application.

Arduino / ESP8266 · Wi-Fi Transmission
3

Real-Time App Display

The mobile app receives live temperature data and displays it clearly — showing the current reading and whether the surroundings are hot or cold with continuous updates.

Mobile App · Live Data Feed
4

Threshold Detection

The system constantly checks whether the temperature crosses a predefined hot threshold. When exceeded, it automatically triggers the cooling response without any user action.

Threshold Logic · Conditional Automation
5

Automatic Fan Control

When the threshold is breached, the relay switch connected to the fan turns ON automatically — the fan starts running and cools the surrounding environment.

Relay Module · Auto Fan Activation
6

Live Cooling Feedback

As the fan cools the surroundings, the app reflects the dropping temperature in real time — you can watch the temperature fall live on screen as the system responds.

Real-Time Feedback · App UI Update
